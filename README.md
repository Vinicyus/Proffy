## Getting Started

Run the API with the following commands.

```
cd Proffy/server

yarn install

yarn start
```
Access API at: http://localhost:3333/


Run the Web App with the following commands.

```
cd Proffy/web

yarn install

yarn start
```
Check out the app at: http://localhost:3000/
